UCup
====

UCup iOS App

Hi! Welcome to the UCup Readme.

This app pulls from the following API/SDK:
-Firebase
-Spotify
-Yelp
-Facebook (to be implemented)

Upon launch, the user will be directed to a screen with 6 options:
  1) Create a party
  2) Find a party
  3) Weather
  4) Pregame music
  5) Food
  6) Feedback
